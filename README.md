# Simulácia objednávkového procesu

Statická ukážková stránka pre univerzitný simulačný projekt optimalizácie objednávkového procesu vo fast-food reštaurácii.

## Technológie

- HTML
- CSS
- JavaScript
- Chart.js cez CDN

Projekt nepoužíva backend, databázu, build nástroje ani platené služby.

## Spustenie lokálne

Otvorte `index.html` priamo v prehliadači alebo použite jednoduchý lokálny server.

## Nasadenie na GitHub Pages

1. Nahrajte súbory do GitHub repozitára.
2. V repozitári otvorte `Settings`.
3. Prejdite do časti `Pages`.
4. V `Build and deployment` vyberte `Deploy from a branch`.
5. Ako branch nastavte `main` a priečinok `/root`.
6. Uložte nastavenie.

GitHub Pages následne zverejní web na adrese vo formáte:

```text
https://pouzivatel.github.io/nazov-repozitara/
```

## Dataset

Tlačidlá na stránke smerujú na:

```text
assets/dataset.xlsx
```

Aktuálny súbor je iba placeholder. Pred odovzdaním projektu ho nahraďte finálnym Excel datasetom s rovnakým názvom.

## Poznámka

Stránka je neoficiálna študentská ukážka. Nepoužíva oficiálne logo ani netvrdí, že ide o oficiálny web reštaurácie McDonald’s.
